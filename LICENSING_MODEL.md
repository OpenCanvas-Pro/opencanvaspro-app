# Licensing Model — OpenCanvas Pro

**Last Updated — 23 February 2026**

---

## Overview

OpenCanvas Pro follows a structured multi-edition licensing model designed to balance transparency, accessibility, and proprietary innovation.

---

## Public Repository

This public repository contains:

- Documentation  
- Conceptual architecture  
- Governance artifacts  
- Example boilerplates  
- Public datasets  

This repository is licensed under the **Apache License 2.0**.

No production engine components, model orchestration logic, automation layers, or proprietary intelligence modules are included in this repository.

The public repository is intended for documentation transparency, community reference, and educational purposes only.

---

## Community Studio

Community Studio is a free-to-use edition designed for individual practitioners, researchers, and small teams.

It includes:

- Cloud-hosted execution environment  
- Guided AutoML workflows  
- Core training and prediction capabilities

The underlying machine learning engine, orchestration logic, automation mechanisms, and decision intelligence components are proprietary and maintained in a private core repository.

Community Studio is distributed under a proprietary license.

It is not open-source.

---

## Journey Edition

Journey Edition is a commercially licensed desktop distribution of OpenCanvas Pro designed for professional practitioners and advanced individual users.

It includes:

- Local desktop execution environment  
- Extended automation capabilities  
- Enhanced robustness and validation tools  
- Expanded configuration controls  

Journey Edition runs locally via packaged distribution (e.g., desktop installer).

Distributed under a proprietary commercial license.

---

## Enterprise Edition

Enterprise Edition provides a customized, mission-critical deployment model designed for regulated environments, industrial operations, and institutional infrastructures.

It includes:

- Local-first execution within the client's controlled infrastructure  
- Autonomous workflow orchestration (EMILIA Autopilot — Enterprise)  
- Robustness validation framework (multi-seed and stress evaluation)  
- Extended governance controls and audit logging  
- Advanced decision intelligence modules  
- Enterprise-grade compliance and traceability mechanisms  

Enterprise deployments are designed to operate within private networks, on-premises environments, or controlled cloud infrastructures defined by the client.

Enterprise Edition is distributed under a proprietary contractual license.
