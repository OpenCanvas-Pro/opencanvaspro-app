# Contributing to OpenCanvas Pro

Thank you for your interest in contributing.

This repository contains documentation, reference material
and example code only.

Before submitting a Pull Request:

1. Fork the repository
2. Create a feature branch
3. Submit a Pull Request
4. Sign the Contributor License Agreement (CLA)

All contributions are governed by the CLA.

## Code Standards

- Follow PEP8 (for example code)
- Use clear commit messages
- Ensure example scripts run without errors

## Reporting Issues

Use GitHub Issues to report documentation errors or suggest improvements.
